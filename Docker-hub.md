<h2 id="docker-hub">Docker Hub</h2>
<pre><code>https://hub.docker.com/
</code></pre>
<p>Docker Hub is a cloud-based repository service that allows users to <strong>store, share, and manage Docker container images</strong>. It is offered by Docker. Developers can package their apps and dependencies into <strong>lightweight, portable containers</strong> using the widely used Docker platform. Applications can then be deployed and scaled more easily since these containers can operate consistently in various environments.</p>
<p>Fundamentally, Docker Hub is a central location where <strong>Docker users can find, share, and work together on containerized applications</strong>. Databases, web servers, programming languages, and a plethora of other software and services are all provided by the extensive library of pre-built Docker images that it hosts.</p>

<h2 id="repository">Repository</h2>
<h1 id="create-a-docker-hub-repository">Create a docker-hub repository</h1>
<ol>
<li>
<p>Sign in to</p>
<pre><code> https://hub.docker.com/
</code></pre>
</li>
<li>
<p>Select <strong>My Hub</strong> &gt; <strong>Repositories</strong>.</p>
</li>
<li>
<p>Near the top-right corner, select <strong>Create repository</strong>.</p>
</li>
<li>
<p>Select a <strong>Namespace</strong>.</p>
<p>You can choose to locate it under your own user account, or under any<br>
organization where you are an owner or editor.</p>
</li>
<li>
<p>Specify the <strong>Repository Name</strong>.</p>
<p>The repository name needs to:</p>
<ul>
<li>Be unique</li>
<li>Have between 2 and 255 characters</li>
<li>Only contain lowercase letters, numbers, hyphens (<code>-</code>), and underscores<br>
(<code>_</code>)</li>
</ul>
</li>
<li>
<p>Specify the <strong>Short description</strong>.</p>
<p>The description can be up to 100 characters. It appears in search results.</p>
</li>
<li>
<p>Select the default visibility.</p>
<ul>
<li><strong>Public</strong>: The repository appears in Docker Hub search results and can be<br>
pulled by everyone.</li>
<li><strong>Private</strong>: The repository doesn’t appear in Docker Hub search results and<br>
is only accessible to you and collaborators.</li>
</ul>
</li>
<li>
<p>Select <strong>Create</strong>.</p>
</li>
</ol>
